Simple Paystub Calculator - Android App

The Simple Paystub Calculator is an Android application designed to help businesses calculate paystubs for employees. The app allows users to enter employee details, compute regular and overtime wages, calculate taxes, and store the information in paystubs. Additionally, it provides functionality to view past calculations and manage employees.

Overview

The application includes multiple activities: MainActivity for paystub calculation, EmployeesActivity for managing employee data, HistoryActivity for displaying paystub history, and AboutActivity for app information.

Features

Calculate paystubs by entering hours worked, overtime hours, hourly rate, and tax rate to compute net pay. Manage employee information with options to add, update, and delete records. View a history of calculated paystubs and delete individual entries as needed.

App Structure

MainActivity handles the paystub calculation. EmployeesActivity lets users manage employee data and view previous paystubs. HistoryActivity displays the history of paystub calculations, and AboutActivity provides basic app information.

Installation

Clone the repository using git clone https://github.com/kabamehmetali/SimplePaystubCalculatorforAndroid. Open the project in Android Studio, build it, and run the application on an emulator or a physical Android device.

Technologies Used

Developed in Java using the Android SDK. The app uses SharedPreferences for local data storage, RecyclerView for listing paystubs and employee data, and Spinner for selecting employees.
