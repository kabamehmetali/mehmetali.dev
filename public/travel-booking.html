P&A Travel Booking Application

P&A Travel is a web-based travel booking platform built with ASP.NET Core MVC and C#. Developed as a group assignment (Group 125), this application simulates a travel agency platform where users can browse and book travel services such as flights, hotels, and car rentals.



Project Description

The platform implements a role-based system using ASP.NET Core Identity for authentication and authorization. Travelers (regular users) can search and book trips, while admins can manage listings and bookings. The project emphasizes a layered MVC architecture, secure user account management, and a responsive interface for a smooth booking experience.

Installation Instructions

Prerequisites: .NET 6.0 or higher and a SQL Server instance (e.g., SQL Server Express LocalDB).
Clone the repository using: git clone https://github.com/kabamehmetali/Assignment_Group_125.git
Open the solution file GBC_Travel_Group_125.sln in Visual Studio 2022 (or later), restore NuGet packages, update the database connection string in appsettings.json if necessary, and run Update-Database if using migrations. Then build and run the application.

Usage

Once launched, users can register or log in. Travelers can browse listings for flights, hotels, and car rentals, make bookings, and view past reservations. Admins can manage travel listings, view all bookings, and manage user accounts.

Additional Information

Developed as a college coursework (COMP2139 assignment) by Group 125, led by Mehmet Ali KABA. The project utilizes ASP.NET Core MVC, Entity Framework Core, and ASP.NET Identity for secure account management.
