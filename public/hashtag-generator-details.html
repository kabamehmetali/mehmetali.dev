Hashtag Generator App

The Hashtag Generator App is an Android application that generates relevant hashtags based on a user-provided prompt. Leveraging OpenAI's GPT-3.5 API, the app dynamically creates hashtags to help boost social media engagement.

Features

Users can enter a prompt to generate hashtags, specify the number of hashtags desired, and copy the generated hashtags to the clipboard with a single tap. The app features a simple and clean user interface.

Setup and Installation

1. Clone the repository or download the source code.
2. Open the project in Android Studio.
3. In MainActivity.java, add your OpenAI API key by setting: private static final String OPENAI_API_KEY = "your-api-key-here";
4. Build and run the application on an Android device or emulator.

Usage

Enter a prompt describing the topic for which you want hashtags, specify the desired number of hashtags, and click "Generate" to fetch hashtags from the OpenAI API. You can then copy the generated hashtags for use on social media.

Dependencies

The app uses OkHttp for network requests and org.json for JSON parsing.
